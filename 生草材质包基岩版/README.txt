Q:有关JE版/网易版移植
A:这个材质包就是从Java版上移植过来的，不如说JE上才是推荐运行的版本
基岩版只是一个分支项目，并且因为版本差异的缘故基岩版可能会缺失一些内容（比如进度）
网易那边的事情我不清楚(ljwy)
如果有那个大佬能把这搞过审的话当然可以搬一下 只要不是付费下载就行 然后别忘了给原版的链接
至于说基岩版的向后兼容是不太可能做的，并且按理说兼容性问题也不大

Q:有关只希望导入部分材质/音效
A:拆包自己研究一下吧，总之解压之后把不想要的东西删掉就好
比如如果觉得声音太吵就把/sounds文件夹删掉
当然主目录的manifest.json pack_icon.png这类的别删就行

Q:有关使用/二创
A:如果是根据这个材质包改动发布（加梗之类的）请保留这个txt。 如果只是来拍视频之类的那不会有太多要求的

Q:有关这个材质包（或者频道）的未来
材质包以后也会持续更新的~然后攒的内容多了可能就会再录一集视频 
不过因为越往后更就会不可避免地内容越杂 也因为许多内容别人都做过了导致很难想出来原创的内容 所以可能后面会缩短一些时长
也有可能就咕了（比如第三弹至少也要三年了）
然后如果你有很好的灵感的话我也可以采纳 如果把你的灵感加入了材质包里我也会在github上的页面里提一下的
然后就是以后我可能会更一些游戏实况类的视频 不一定是MC 如果大家能来资瓷一下的话就特别感谢了~
